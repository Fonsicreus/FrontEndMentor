## Overview

### The challenge

Users should be able to:

- View the optimal layout for each of the website's pages depending on their device's screen size
- See hover states for all interactive elements on the page
- View each page and be able to toggle between the tabs to see new information

I used the AI to help me with the initial setup, including Tailwind configuration and project structure and ideas for the JSON data file. I also used the AI to help me fix the errors I was getting in the console.
